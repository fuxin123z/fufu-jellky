Getting Started

An introduction to Heeney - Modern Blog HTML5 template, folder structure, template set up and everying you need to know before you start customizing it.

All you need to know about how to properly set up the Heeney - Modern Blog HTML5 template.

This template is a responsive and retina-ready HTML5 website with bootstrap grid system layout. It is optimized for mobile touch and swipe. Heeney is the perfect template for magazines, blog, personal blog or freelancers. It is light weight making it a very fast and responsive template.

Installing the Template

1. To begin, it is very important to understand that you must either have your own server space or use someone else’s. The server space can be purchased at many different places around the internet. You need somewhere to put the files that you purchased so others can find them online.

2. After downloading your purchased item from ThemeForest you'll get a .zip archive, make sure you unzip it.

3. After you've unzipped the archive you'll find two directories in there, one will be the directory in which this Documentation is located, and the other one would be the template itself.

4. Now that you have your template downloaded and you're ready to go, open your FTP client (WinSCP, FileZilla or any other FTP clients) and upload the content inside the "heeney" template directory.

5. To login into your FTP you're gonna need some credentials, here comes in play what we've specified above, a host. All hosting providers will provide you the FTP credentials for your account, those will contain the address (ftp.yourwebsite.com), username (the name they've set for your account), and a password (which in most cases is a highly secure autogenerated password).

6. Use those credentials to login into your FTP and simply drag & drop the content of the "heeney" directory.

7. That's it, you're now set, from now the only thing that remains would be to adjust and modify the template as you see fit.

Adding a Blog Post

To add a blog post inside your Blog Grid first make sure you're adding the code bellow inside the <div class="blog-items"></div> that wraps your items. Basically you want to add the code inside that blog-items div which will serve as the container for your Blog items.

<div class="col-lg-6 col-md-6 col-sm-6 col-12">
	<div class="blog-item">
		<div class="blog-img">
			<img src="images/resources/item7.jpg" alt="">
		</div>
		<div class="blog-info">
			<a href="#" title="" class="post-category">Beauty</a>
			<h3 class="post-title"><a href="#" title="">Diddy Offers Intimate Look at His Birthday Beauty Routine </a></h3>

</div>
